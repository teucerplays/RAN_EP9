move
You can move the character by clicking the left button of mouse in area you can see
This game supports 'path finding' system which allows the character to move a target location automatically even if the path is not straight.
If you have a teleport card, you can move a certain place faster. Followings are the usage
-School Teleport Card : Teleport to the your School main building inside
-Start Teleport Card : Teleport the character to start location.
-Last Teleport Card : Teleport to the last location the teleport card is used or the player is dead
-Friend Teleport Card : Teleport to the current location of your target friend
bus
The bus passport allows player to move directly another place by bus even if there is the loading place in your path.
Teleport(connect) card
"If you have a teleport card, you can move a certain place faster. Followings are the usage

-School Teleport Card : Teleport to the your School main building inside
-Start Teleport Card : Teleport the character to start location.
-Last Teleport Card : Teleport to the last location the teleport card is used or the player is dead
-Friend Teleport Card : Teleport to the current location of your target friend"
battle
Player can attack the enemies except the party menber and NPC; There are 'Normal Attack' by clicking left button of mouse and 'Skill Attack' by clicking right button of mouse after selecting skill.
battle/non-battle area
All fight actions including holding the weapon in non-fight area are disabled
how to autoattack
Click right-button of mouse with pressing 'Ctrl' key after you located your cursor on target to be attacked
You can attack the target automatically by a single click once the 'Force attack' option is toggled in 'Game' menu of 'Option' which is called by hitting 'ESC' key.
Skill
It refers a special attack or defense skill which can be learned from 'Skill document'. To acquire the skill, the requirement of each skill should be satisfied.
how to Get(Learn) skill
'Skill document' can be acquired from the quests or NPC
When the mouse cursor is located on 'Skill document' in inventory, the requirements are displayed.
(If you do not meet one or more of the requirements, those requirements will display in red text)
If you meet all requirements of target 'Skill document', then you can learn that skill by Right-Clicking on 'Skill document'.
But you cannot unlearn the skills that you already learned
Skill level up
All skills can be upgraded up to maximum 9 levels
'Skill window' is activated by using the shortcut key (K). The requirements for next level is displayed when the cursor is over target skill.
If all requirements are satisfied (displayed in green text), the skill can be upgraded by clicking '▲' button in right side
(If the skill points is not enough, the upgrade cannot be done)
But, be careful with upgrade You cannot undo the applied skill points.
how to use skill
'Skill window' is activated by hitting the shortcut key (K).

1. Drag the favorite skill icon(frequently used skill) to 'Skill Shortcut Window' at left side of screen by dragging skill icon


(If you cannot see 'Skill Shortcut window', press '`' key)

2. To select the target skill, hit the shortcut key '1~0' or left-click on that skill

3. Cast the skill by Right-Click on target object

Skill tray can be shifted by hitting F1~F4
Use item
The items gathered by the battle or trade is used by its usage
Item Grade
- General Item
It refers to the item that has the basic states

- Rare Item
It refers to the upgraded item that contains the extra or special features beside basic states.

- Unique Item
It refers to the special item that provide more features than Rare Item. It is hardly found in game.

- Quest Item
It refers to the item that can be used only for quest and cannot be used arbitrary
Item Type
-Consumable Item
It has the characteristic that it can be consumed by using it. It can be purchased in shop easily. It saved in inventory automatically when you pick them up.

1. Activate the Inventory window by using 'shortcut (I)'

2. Move the frequently used item into 'Item Shortcut window' in upper left side of screen by using left mouse button

(If you cannot see 'Item Shortcut window', press 'Tab' key to activate it)

3. You can access the items in shortcut window by 'Q,W,E,A,S,D' button

- Non-consumable item
It is the item that is not destroyed or consumed by using it like the weapons and armors. It cannot be access by 'Item Shortcut window'
Item Equip
Get the equipments(can be equipped)

Activate the Inventory window by using 'shortcut (I)'

1. If the slot for the target equipment to be equipped is empty
After selecting the target equipment by left-click, move it to the target slot and then click the left button of mouse.

2. If the same type of equipment is already placed in slot
After selecting the target equipment by left-click, move it to the target slot and then click the left button of mouse. Once it replaced with old equipment, move and drop the old equipment with left-click
Upgrade
The most of weapons and armors can be upgraded
If the upgrade is successful, the increase of attack and defense power will be very helpful. But any failure of upgrade might remove all upgraded states and it even destroy the item.
How to upgrade
If the item need to be upgraded is equipped, it should be unequipped and be placed in inventory. (It cannot be upgraded while it is equipped)

1. Select the upgrade ingredient item (Black stone, Coating material, and etc) by using Left-click

2. Move the cursor to the target item to be upgraded and click the right button of mouse for upgrading

3. The result of upgrade is displayed in chat window. If successful, the new states can be checked by locating the cursor on item.
NPC
Unlikely the object that attack the players, NPC provide the utilities of game such that providing the information and quests, selling and buying the items, and accessing the locker
Locker
Once you go to Locker NPC and talk with him by left-click, you can access the locker by 'Use Locker' menu
After selecting 'player Locker', you can use your warehouse which allows to keep more items in game.

1. Personal Locker
If 'Locker Expansion' is activated, the space of Locker become double.

2. Club Locker
The higher rank you get the more space you will get
Store
If you go to Shop NPC and talk with him by left-click, you can access the shop by 'Sell/Buy Item' menu
After selecting 'Sell/Buy Item', the shop item window and player inventory will be opened and the item transaction can be performed..

Buying

Select the wanted item in 'Item list' in shop item window by using left-click

1. If the money is enough to buy the item, then the selected item is moved into inventory and the price of that item is deducted from your money

2. For consumable item, the quantity menu is displayed

3.Once the quantity of item is entered and the money is enough, then same amount of items are saved in inventory


Selling

After doing left-click to select the desire item to be sold, move it to 'Item list' window. It automatically compute the price and sell the item.
Private Shop
What is private shop system?
It is the system that allows the transaction between players in any place.
To open the private shop, the permission is required from the school

How to use private shop
'shop open permission' is required to create a private shop
'shop open permission' can be purchased in shop
Right-click the permission item in inventory to open the shop menu
Set the name of shop
You can enter maximum of 40 letters for English and 20 letters for Korean
Move the items into private shop window like buying a item procedure. Maximum of 24 items can be registered.
Once 'Start' button is clicked, the shop is opened for everyone. And the character cannot move until the shop is closed.
(You cannot change the contents of shop after it opened)
Although the permission only allows to open the shop a limited number of times, there is no time restriction after creating the private shop.
If the shops are opened, the name of shop is displayed. The desire item to be purchased can be brought with same procedure as buying a item from NPC
Startpoint
If you see this choice(sentence? Gate?), you can select the start point after the death in battle.
Duel
It is the system that allows the players to compare their characters with a fight. The battle is processed upon the mutual agreement
How to dual
Move the cursor to the character you want to fight

1. Click the left button of mouse while pressing 'Shift' button. Then the menu that contains 5-6 Icons is displayed.

2. Activate 'Duel setting menu' by selecting 4th icon from left

3. Click 'Confirm' button after changing the duel setting.

4. It goes in duel mode once the opponent player clicks 'Confirm' button

5. The duel is started with 'Fight' message on screen
Dual rule setting
- Medicine use
The maxinum number of medicine the players can use in duel can be set. By default, it is '0'

-Health Set
Before the start of duel, the character health can be adjusted with certain amount of rate. Each player can use the different ratio.

-Area Limit
The 'Ring' is displayed around duel ground if the area limit is toggled. Once the player move outside of 'Ring' then he/she will lose the duel.
Dual Procedure
After entering 'Duel' mode upon the mutual agreement, 'Duel' is continued until one player HP become 0 withing 180 secs and all invitations of another duel will be discarded.
Interface
In game screen, there are several menu window. Each window can be moved and resized.
Character window
By pressing shortcut key (C), the information of character can be checked. See the state description for the detail
Party window
Activated by using shortcut (P). If you are in party, you can see the information of party member.
The information about the skill effect and location is displayed
Inventory window
Activated by using shortcut(I). The most of item can be deposited. And the weapons and armors can be equiped through this window.
The upgrade of item is processed in here
Skill window
Activated by using shortcut(K). It displays the player skills and their requirements. The skill upgrade can be done.
Chatting window
As you login the game, it is already activated in left bottom of screen. Its purpose is the communication between players. Using 'Enter' key, you can send message.

- All :Show all messages in game like the notice and chatting
Once 'All' button is toggled, the detail option can be set like 'General Chat', 'Whisper Chat' , 'Party Chat',and 'Club Chat'.

- Whisper : Show only the whisper messages
To send a whisper message, your can use 'Friends' window or start with '@CharacterName' in message window
Also by double clicking the previous message, you can directly send a whisper message.

- Party : Show only the message between party members in chatting window
Append '#' at the head of message to send party message

-Club : Show only the message between club members in chatting window
Append '%' at the head of message to send club message

-Alliance : Show only the message between alliance member in chatting window
Append '!' at the head of message to send alliance message

-System : Show only the game notice message in chatting window

- If 'Megaphone' is in inventory, send a message to everybody in same server by starting with '$'
Trading window
It is the window that is activated during the trade between player. player can provide the item or price in window and the transaction is completed with the confirmation between players.
For the safe transaction, please check the price and item information again before confirmation.
Resurrection window
In battle, the character may die for several reason
When the character died, this window is activated automatically to decide how to resurrect
After the resurrection, the character is revived and placed in starting point or certain saved place. The part of experience will be deducted starting from level 10.
If player use 'Spirit Sphere' then player can revive on the place you died without the experience deduction.
But if you are revived by 'Resurrection' skill, the experience is still deducted.
Mini map
Activated by shortcut (M).
This window indicate the location of character and NPC as well as the party number if the character is in party.
It can be moved and resized.

Game End
To exit the game, the system menu can be activated by 'ESC'
After the system menu is activated, confirm the game end again.
Ability Description
The several information is saved in the character and while playing, the character will grow up with some state changes
Name
It is the unique name of character. It cannot be changed after character creation
And the slang and insulted words cannot be use for names
Department
This indicates the class which character belongs and currently Brawler, Swordsman, Archer, Shaman, Extreme and Science are prepared as default classes.

-Brawler : For Melee fight. Advantage on fast attack speed and high evade rate
-Swordman : For Melee fight. Less attack speed but high defense
-Archery : For Ranger fight. High attack power with long range
-Shaman : For Melee and Range fight and support. It can be used several ways
-Extreme : Hybrid combat type (Close range+Long range), fast attack speed, strong attack power and long range attacks.
-Science : Long rage assist type, a little weak power but various special effects.
Level
It can move to next level by accumulated experience. Every level player will get 3 state points and 1 skill points which is used to create unique character.
Power
Influence on the melee and range attack power
It is very important state for melee character like brawler and swordman
vitality
Influence on maximum HP (Health point)
intelligence
Influence on maximum MP and Magic power
It is important state for Shaman which uses the magic attack

Dexterity
Influence on defense, range attack power, magic attack power, and melee attack power
The effect rate is depended on class
Stamina
Influence on SP
If Stamina is low, the attack rate and power can be decreased by maximum of 50%.
HP
It refers to Health point. If it become '0' or below '0', the character will be died.
The food and medicine can recover HP and a small amount of HP is automatically healed every second.
The auto healing rate can be increased by using or equipping some items like 'Healing drink'
MP
It refers to the amount of spirit which is used for casting skill.
The food and medicine can recover MP and a small amount of MP is automatically healed every second.
The auto healing rate can be increased by using or equipping some special items
SP
It refers to amount of stamina which affect on the attack power and rate.
The food and medicine can recover SP and a small amount of SP is automatically healed every second.
The auto healing rate can be increased by using or equipping some special items
EXP
It is the experience point the character receive from the battle.
By accumulating the experience, the character can do leveling up

Experience points can be received by following cases
- kill the monsters whose name are red color
- Complete the quests which gives the experience points
- receive the 'Exp Get' effect from 'Question Box'

Experience points can be deductive by following case
- Die when the character level is higher than or equal to 10
Fight level
Effected by 'Power' and 'Dexterity'. Influence on Melee attack power
Shooting Power
Effected by Dex and Power, Ability concerned with Shooting Attack Power.
Attack Power
Effected by basic stats and equipped weapon of a character,
the amount of damage can be given to opponent during the combat.
Defense Power
Effected by basic stats and equipped shielding of a character,
the amount of defense value against the opponent's attack during the combat.
Resistance Power
Resistance level against the magic with traits, the existing resistance are None, Fire, Cold, Electricity, Toxicity.
If the resistance level is high then better defense against the specific trait and also the resistance to the abnormal condition.
Class Info
Currently prepared 6 Classes have different abilities and growth range and so the game proceeding cannot be the same.
Brawling
Close range attacks using arms, legs, head and so forth of all the body parts, speedy and continuous attack with plenty of stamina. As the growth type it can be diverted to fist type, leg type and qigong type.
Swordsmanship
Middle or close range attack using the knife and it has lots of stamina and good defense power.
As the growth type it can be diverted to sword type, blade type and qigong type.
Archery
Long range attacks using the bow, has high accuracy rate and attack power.
As the growth type it can be diverted to speed bow type, strong bow type and qi bow type.
Shamanism
Use Magic and Close, Middle and Long range attack can be possible by the growth type.
Extreme
Hybrid combat type which uses every weapons without Science's.
As the growth type, character can be raised as ‘Melee’, 'Archery’ or ‘Magic’ fighting type.
And Extreme can switch to use two types of weapon.
Science
Long range combat type which uses guns with various effects.
As the growth type, character can be raised as ‘Debuff’, 'Restriction’ or ‘Summon’ fighting type.
Party
When a party is formed, extra EXP can be acquired, easier to play the game.
The EXP acquiring standard will be set by and to the lowest level in the party and EXP distribution differs by the level.
But, as the level difference gets larger acquired EXP amount would be decreased.
Forming Party
Party Master places the cursor on the player he or she wants to recruit to party.

1. Display the 5 to 6 icons enlinked menu by clicking left mouse button while pressing the 'Shift' button.

2. Select the icon located on third from the left to send the invitation message.

3. Upon the target player's acceptance, player will be joined to party. But, the target player must not be in any other party.
About party progress
'- Party members can share the information and can be form with 2 to 8 people.

'- When members are in certain distance the exp acquired by a member in a combet will be shared.

'- When a member has moved to a different map, he or she would still be in the party list but displayed in grey and that specific member info wouldn't be renewed.

'- To withdraw from the party, press the withdraw button in the party menu which can be open with 'P' button.

'- Party master can use the party menu which could be opened with shortcut key (P) for banning party members or disband the party.
Party Disband
- Voluntary Withdrawal
: When party member has withdrew from the party by one's sole intention.

- Forced Withdrawal
: Disbanded without ones’ intention by the Party Master and its rights.

- Log-Out
: Status of logged out from the game due to the ones’ PC problem or one’s sole intention.

- Automatically disband when the party master has withdrawn or only one of party member is left.

Club
Maximum 60 member can be joined in a club. Possible to participate directly in the Lead Club Battle.
Establishing Club
-Form a party of 5 or more members
:Concerned Party Master apply to each school's head teacher in teacher's room. (Ask about Club -> Create a Club)

-Newly created club can have 15 maximum members and as the club rank gets higher the number of member who can be accepted will be increased to maximum 60.

*Terms of club registration
- When Party Master life points are 0 or higher.
- When Party Master is Level 30 or higher.
- When all the Party members are not in any other clubs.
- When the applicant can afford the fee
Club Mark Change
?' mark will be displayed when club is just established but with the rights given to club master, club mark can be created and registered with club mark modify button.
Mark's size must be 16X11, and it must be in 24bit BMP or 32bit TGA type. Be sure to input the file extension when entering Club Mark file name.
Club Chat
Starting a message with % in front will only be sent to the members of the club and clicking member name in the club member list enables you to send whisper.
Joining Club
Only the Club Master has the rights to use.

possible to send invitation to other characters to join one's own party and character will be joined upon agreeing to it.
Dismiss / Withdraw
General members can withdraw and club master can dismiss a club member and character who withdrew or dismissed in this way can not join a different club for 1 day.
But, creating a new club is possible.
Disband
Only the Club Master has the rights to disband the club and when it is carried out, 7 days are given to disband from the moment it was executed.
It is possible to cancel the disband during this 7 days.
Be aware that club Master will loose the EXP as same amount as when died for disbanding the club.
Alliance
Maximum 3 Clubs can be allied.
Alliance menu in the club menu will be working and the allied club will be displayed.

To make an alliance,
from the menu displayed when one club master shift + click on a different club master.
Press 'Invite to Ally' icon and just agree.

Lead Club Battle
Since the 4 schools were established, Saint Foundation has been focused on making all the financial status and all the other managing work to be automated and all the functions were centralized to each local data processing dept.
As the phenomenon occurred, all the data processing dept stopped operating but it started to recover its functions by the repeated restoration work done by the students.
Therefore, the ignored taxes were imposed again by automatic transfer from the each one's account and 3 schools and Sungmoon Dong, Hyunam Dong, Bonghwang Dong and Sang Dong are the areas effective.
Each region's data processing depts. are supposed to be managed by a person with the special rights and the teachers of 3 schools planned to use this as the chance to give motivation to students' self study and sense of belonging, so the 'Lead Club' system was enforced.
How to Play
Day & Time : Every Sunday from 20:00, for 1 hour.
Location : Each Regional Data Processing Dept. - Only the bus can be used to get to the location for safety reason.
Limitations : Not possible, to be a lead club in more than one region at the same time.

-It will be announced 1hour, 30minutes and 10 minutes prior to the Battle.
-Lead club members can enter the applicable region's data processing dept freely until the rights of lead club is still standing.
-At 20:00 hours, the battle begin in each region's data processing Dept with the notice.
-At the same time in the regional data processing Dept., unlimited PK mode will be started between the different club members
-If the attack can validate for one minute at the validation terminal the attack is switched to defense and all the other club members will be forced out to data processing dept.
-Validation can only be done by the Club Master who has the validation CD
-The club on defending side will be the lead club when the time is up.
-No EXP penalty will be applied when killed during the Lead Club Battle.

->If the battle ends in draw then the commercial rights will be handed over to the Mob and all the players will be charged of 5% trading fee.
Managing Fee
-Club which became the winner of lead club battle will maintain the rights until the next lead club battle is over.
-Club master who acquired the rights in one area among 4 must set the trading fee in between 0~50%. (Basic 5%)
-Each regional trade fees are collected in every hour and delivered to the lead club's locker automatically.
Regional Data Processing Dept.
Starting Point can be changed at the Data Processing Dept. entrance but not possible in inside.
Validation CD
Can be purchased at the entrance of each Data Processing Dept. But, self-created pigments were used to make the CD instead of the normal materials, so the durability is very short (1Day)
